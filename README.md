# ttsui-mac

Wrapper around [Qwen3-TTS](https://qwen.ai/blog?id=qwen3tts-0115). Completely vibe-coded. Assisted by GLM-5.

## Run

### Prepare Python environment

1. Install dependencies

```bash
pip install mlx-audio
pip install numpy
pip install soundfile
```

2. Configure `TTSUI_PYTHON` environment variable to point to your Python interpreter.


### Run in Xcode

Open project in Xcode and hit run.
